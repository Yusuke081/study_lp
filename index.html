ABOUT

Juitaについて

Juita Web Designは、インドネシア（ジャカルタ含む） バリ島にてホームページ制作業務を中心にフリーランスとして活動しているWebデザイナー事務所（SOHO）。

デザイナーは、大学（日本語学科）を卒業後、日本へ留学。 その後、Webデザイン専門学校（神奈川県横浜市）へ入学し日本のWeb知識、スキルを2年間学ぶ。Webデザイン専門学校卒業後は、アパレル会社（神奈川県藤沢市）にWeｂデザイナーとして勤務。その傍ら、日本で起業、開店する外国人を主にWebデザイナーとしてサポート。その後、出版会社（東京都千代田区）にWebデザイナーとして勤務、ジャカルタ支社たち上げの為、支社長としてジャカルタへ赴任。その後、バリ島にてJuita Web Designをたち上げる。
